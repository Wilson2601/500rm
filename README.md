# RiseRight - Smart Sleep Discipline App

## Overview
RiseRight is a mobile application developed for university students to overcome morning lateness through gamification and behavioral technology.

## Features
- **Smart Alarm:** Challenge-based wake-up system requiring QR verification.
- **Sleep Lock:** Restricts social media usage during designated sleep hours.
- **Gamification:** Reward system to track progress and consistency.

## Technical Implementation
- **Frontend:** HTML5, CSS3, JavaScript.
- **Storage:** LocalStorage for persistent user data.
- **Hardware Integration:** Browser-based camera API for QR challenges.
